Simple Dropdown Menu

This example demonstrates a dropdown menu using HTML <select> and <option> tags with enhanced styling.

How It Works

  • The dropdown is created using the HTML <select> element
  • Each option is defined using the <option> tag
  • JavaScript is used to display the selected value
  • CSS enhances the appearance and adds interactive effects
<select id="fruit-select">
  <option value="" disabled selected>Select a fruit...</option>
  <option value="apple">Apple</option>
  <option value="banana">Banana</option>
  <option value="orange">Orange</option>
  <option value="grape">Grape</option>
</select>
